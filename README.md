thibaudanthoine.com
===================

My personal website.

Using Silex, HTML5, CSS3, Javascript, Node, Grunt, Sass, Compass.

### Installation

1 - Install bower and grunt-cli:

```
npm install -g grunt-cli
```

```
npm install -g bower
```

2 - Install bower and node dependencies:

```
bower install
```

```
npm install
```

3 - Launch Grunt:

```
grunt
```

thibaudanthoine.com
===================

My personal website.

Using Sass, Compass, Node and Grunt.

### Installation

1 - Install bower and grunt-cli:

```
npm install -g grunt-cli
```

```
npm install -g bower
```

2 - Install bower and node dependencies:

```
bower install
```

```
npm install
```

3 - Launch Grunt:

```
grunt
```
